# Next.js 15 Project

This is a fresh Next.js 15 project initialized with React 19, TypeScript, Tailwind CSS, and shadcn/ui.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## Folder Structure

- `/components` - Place all reusable UI elements (including shadcn/ui components) here.
- `/app` - Place all page routes and application layout here.
