# GGH — Gomla Go Home
> جملة لحد البيت
> *Wholesale groceries, delivered to your doorstep.*

GGH is a wholesale grocery marketplace built for Egyptian households and small businesses. It brings the wholesale experience online, allowing customers to browse household staples at wholesale prices, order in bulk quantities, and receive delivery at home.

For project documentation, architecture, and coding standards, refer to the `docs/` folder.
